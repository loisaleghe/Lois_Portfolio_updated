Vocabulary Builder

Motivation

To create an app that would allow a user to enter a word, find its definition, identify synonyms and antonyms for the word, and show related words in a bubble diagram.
The purpose of this project is to help improve an individual's vocabulary by providing various definitions of an inserted word along side at most ten synonyms for better understanding.

Tech Stack

JavaScript, HTML, CSS, React, Meriam Webster Thesaurus API, Meriam Webster Dictionary API, Datamuse API
